Richard's Riemannian Manifold Talk at Microsoft Research

At the very beginning, we present a hierarchy of manifolds for the sake of easier explanation.

  1. Continuous manifold: One can define continuous functions.
  2. Differentiable manifold: One can define differentiable functions.
  3. Riemannian manifold: One can define depths and angles.

In this article, we focus on the Riemannian manifold. For more detailed explanation, please refer to [1].

We present an intuitive definition of a manifold. We may think of a manifold as a smooth surface in . Then every point on the manifold (surface) can be seen as a Euclidean space of . Please note that a Euclidean space is a manifold by itself.

In order to define a Riemannian manifold, we first need to define an inner product and a Hilbert space. For the sake of simplicity, a Hilbert space is a vector space within which the inner product is defined. Usually, when we talk about a Hilbert space, we think of it being infinite dimensional.

We here intuitively understand norms and inner products. A norm is a distance function defined on a vector space, and an inner product allows one to measure angles and distances.

Now we give the definition of a Riemannian manifold, which is a manifold with an inner product defined in the tangent space at each point, so that we can measure angles at each point. We define the “geodesic distance” to refer to the length of curves. A geodesic distance is a local curve with the shortest distance. Taking the great circle as an example, the shortest distance is the dense red curve, whereas the geodesic is the dashed red curve since at every point the dashed red curve is local minimal.

We present more descriptions of a geodesic.

  1. A locally distance minimizing curve.
  2. A curve on a surface whose acceleration is always normal to the surface.
  3. A taut piece of elastic band on the surface.

Personally, at an instantaneous moment, the direction of a geodesic is the same as the marching direction. For example, in order to plot a geodesic of a cone, we can put a paper with a point being able to draw underneath a cone and roll the cone while keeping it relatively static with the paper. Then that drawing point will draw a geodesic on the cone since that point on the cone instantaneously always holds the same moving direction.
