Richard's Riemannian Manifold Talk at Microsoft Research

2019.07.23 Zhenyue Qin Learning Notes

At the very beginning, we present a hierarchy of manifolds for the sake of easier explanation.

  1. Continuous manifold: One can define continuous functions.
  2. Differentiable manifold: One can define differentiable functions.
  3. Riemannian manifold: One can define depths and angles.

In this article, we focus on the Riemannian manifold. For more detailed explanation, please refer to [1].
