make_template is to create a function. In general, the functions created by make_template are mutually unique, with an exception that two functions created by make_template share the same scope and is redefined when reentering the same scope. Another special case is when have utilized the argument unique_name_, by which the same template will be accessed when the same unique_name_ is provided.

*args addresses unnamed arguments and pass them as a list. **kwargs handles named keywords and interpret them as a dictionary, where keys are variable names and values are actual variable values.
